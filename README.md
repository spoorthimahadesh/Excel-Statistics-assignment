📊 Statistical Data Analysis using Excel
📌 Project Overview

This project demonstrates statistical data analysis using Microsoft Excel to understand relationships between business variables. The analysis applies statistical techniques such as mean, average, regression analysis, and R-squared interpretation to identify trends and patterns in the dataset.

The project focuses on using Excel’s built-in tools to perform data exploration, statistical modeling, and visualization, helping derive meaningful insights that support data-driven decision making.

🎯 Objectives

- Perform basic statistical analysis using Excel

- Understand relationships between variables using regression analysis

- Interpret R-squared and regression coefficients

- Visualize relationships using scatter plots

- Draw meaningful conclusions from statistical results

📊 Analysis Performed

The following statistical techniques were applied:

- Mean and Average calculations

- Regression analysis using Excel Data Analysis ToolPak

- Interpretation of R-squared values

- Coefficient analysis

- Scatter plot visualization for variable relationships

🔍 Key Insights

- Regression analysis helped identify relationships between independent and dependent variables.

- R-squared values were used to measure how well the model explains variations in the data.

- Scatter plots provided visual confirmation of correlations between variables.

🛠 Tools & Technologies

- Microsoft Excel

- Excel Data Analysis ToolPak

- Statistical Analysis Techniques

- Data Visualization
